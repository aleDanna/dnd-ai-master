    Apply structuredClone immutability (same discipline as applyEvent). Switch on event.type.
    Reducer effects (per CONTEXT §"Event schema (LOCKED)" — ALL effects are locked):
      - combat_start: return { active:true, round:1, currentIdx:0, turnOrder:[], monsters:[] }
        (re-initializes; discards any prior encounter state — fresh start per CONTEXT).
      - monster_spawn: if not active, skip (return state unchanged — defensive); append new monster
        { id, name, hpMax, hpCurrent:hpMax, ac, initiativeBonus, isAlive:true, conditions:[] }.
        Duplicate id: skip (deterministic replay must be idempotent on duplicate spawn events).
      - initiative_set: if not active, skip; set turnOrder = payload.order (copy the array); reset currentIdx=0.
      - turn_advance: if not active, skip; newIdx = state.currentIdx + 1;
        if newIdx >= state.turnOrder.length: currentIdx=0, round=state.round+1;
        else: currentIdx=newIdx. Edge case: turnOrder is empty — skip (return state, don't throw).
      - monster_hp_change: if not active, skip; find monster by id;
        if not found, skip (defensive — out-of-order events are ignored per CONTEXT §"Claude's Discretion");
        newHp = Math.max(0, monster.hpCurrent + payload.delta);
        isAlive = newHp > 0;
        return state with updated monster (use map to replace by id).
        Healing a dead monster above 0 sets isAlive=true (additive delta model — from spec §"Error handling").
      - combat_end: return { ...state, active:false }.
      - default: return state (unknown encounter event — future-proof skip, no throw).
