- **Issue:** The LOCKED block's worked example contains the prose word "Luffy" (in the BAD example "Luffy si lancia..."). The pre-existing Phase 02.1 test asserted roster ordering via bare `out.indexOf('Luffy')` / `indexOf('Usopp')`.
