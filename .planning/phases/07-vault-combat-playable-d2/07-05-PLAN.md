Root cause (validated 2026-05-28 via the `_probe-combat.ts` diagnostic): the vault path's models (qwen3:30b AND gemma4) anchor to the conversation pattern over the system prompt (recency bias). Output: a pure `buildTurnDirective` helper + its wiring into the turn route's vault branch (append to the last user turn) + unit tests. NOT part of buildVaultSystemPrompt â†’ REQ-022 untouched.
